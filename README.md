# Databricks Change Data Capture Integration with AWS RDS

## Overview

This repository documents a comprehensive technical implementation guide for establishing a Change Data Capture (CDC) pipeline between an AWS Relational Database Service (RDS) SQL Server instance and a Databricks Lakehouse platform deployed within the AWS ecosystem. The implementation addresses the architectural complexity inherent to multi-VPC deployments, providing detailed configuration procedures for database-level CDC enablement, inter-VPC network connectivity, and secure data ingestion workflows.
